### Java 基础

1.  夯实 Java 编程基础 -- **《Java 编程思想》**
2.  完全没有 Java 编程基础 -- **《Java 核心技术》**
3.  设计和实现高质量代码 -- **《Effective Java》**
4.  设计模式 -- **《Head First 设计模式》**

1.  先理论，后实践 -- **《Java 并发编程实战》**
2.  国内最好的 JVM 书籍之一 -- **《深入理解 Java 虚拟机》**

### 性能优化

1.  体会其思路和原理 -- **《Java 性能优化权威指南》**
